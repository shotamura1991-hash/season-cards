# きせつカード（季節カード学習アプリ）

小学校受験でよく出る「季節（行事・植物・食べ物など）」を、カードで楽しく覚えるWebアプリです。

- カードの絵を見て、まず **なまえ** を4択から選ぶ
- 正解したら、次に **きせつ（はる・なつ・あき・ふゆ）** を選ぶ
- 両方正解で次のカードへ（まちがえても、正解するまで挑戦できます）

PC・iPad・スマホなど、ブラウザがあればどの端末でも使えます。

## つかいかた（パソコンでためす）

`index.html` を **ダブルクリック** するだけ。ブラウザで開いて遊べます（インターネット接続も不要）。

## カードのふやしかた

`cards.js` の `CARDS` に行を追加するだけです。絵文字でもOK：

```js
{ id: "yukigassen", name: "ゆきがっせん", season: "fuyu", emoji: "⛄" },
```

- `id` … 他と重ならない名前（英数字）
- `name` … カードのなまえ（ひらがな）
- `season` … `haru` / `natsu` / `aki` / `fuyu` のどれか
- `emoji` … 絵文字を入れると、その絵がカードに出ます
- じょうずに描きたいときは `emoji` の代わりに `art:` にSVGを書いてもOK（既存カードが見本です）

## ファイルの説明

| ファイル | 役割 |
|---|---|
| `index.html` | 画面の骨組み |
| `style.css` | 見た目（色・大きさ・レイアウト） |
| `cards.js` | カードのデータと絵（★ここを編集して追加） |
| `app.js` | クイズの動き |

## インターネットで公開する（GitHub Pages）

このフォルダをGitHubに上げて、GitHub Pagesをオンにすると、URLでどの端末からも開けます。
手順は会話のメッセージを参照してください。
